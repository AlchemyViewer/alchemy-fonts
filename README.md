# Alchemy Viewer Font Repository

This repository contains fonts utilized by Alchemy Viewer.

The woff2 format fonts were created via woff2_compress avaliable via most package managers as `woff2`

* Inter            - 4.1    - ttf->woff2       - https://github.com/rsms/inter
* DejaVu Sans      - 2.37   - ttf->woff2       - https://github.com/dejavu-fonts/dejavu-fonts
* Source Han Sans  - 2.005R - (OTF TTC)->woff2 - https://github.com/adobe-fonts/source-han-sans
* Twemoji-SVG-Font - 17.0.2 - ttf->woff2       - https://github.com/AlchemyViewer/twemoji-color-font
